Parameters

text

Type: string

The source text from which the substring will be returned from.

startPosition

Type: Nullable<int>

An integer position to start the search from. If no starting position is specified, an index position of 1 (base 1) is used.

prefixText

Type: string

A string that is used to identify the starting location for the text to return. The starting position will be advanced to this search text. If this prefix text is not found, nothing is returned.

suffixText

Type: string

A string that is used to identify where the returned text is to stop. If this suffix text is not specified, the remaining text in the string is returned. If the suffix text is not found, nothing is returned.

nextPosition

Type: int

Returns the character index position that follows after the suffix text. If no suffix text was specified, Len(text) + 1 is returned.
